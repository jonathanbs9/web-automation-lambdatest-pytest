# web-automation-lambdatest-pytest

## Dependencies
poetry
autopep8 
flake8 
pyleniumio 

## Install Poetry
(env) ```pip install poetry```
## Create poetry
```poetry init```
    * Add pyleniumio
    * Add autopep8
    * Add flake8

```poetry install```

## Initialize Pylenium
```poetry run pylenium init```

## Run test
```pytest```
